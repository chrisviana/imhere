# Projeto I'm Here

Projeto desenvolvido durante um curso de React Native

O aplicativo tem por proposta criar uma lista de pessoas que estão presente em algum evento e também pode remover as pessoas adicionadas.

Foi utilizado:
- React Native
- Expo
- TypeScript
